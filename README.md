## ⌨️ Using the Application

POS-fr uses a simple button-based layout for quick navigation during sales.

### Recording a Sale
Click the New Sale button on the main dashboard. Scan an item barcode or select the product from the list. The software adds the item to the receipt. Click Pay to finish the transaction. You can choose cash or card as the payment method.

### Managing Inventory
Select the Inventory tab to view your current stock. You can add new items by clicking the Add Product button. Type the name, price, and current stock count of the item. Click Save to log the information in the database. The software updates these numbers every time you complete a sale.

### Creating Reports
Click the Reports tab to see your daily sales totals. Select a date range to view your profits for the week or month. You can export these reports as a text file for your records.

## 📝 Frequently Asked Questions

Do I need an internet connection to use POS-fr?
You can use the software offline for most tasks. You only need the internet to download updates.

Can I print receipts?
Yes. The software supports most standard thermal receipt printers. Connect your printer to the computer via USB before you open the app.

Does this save my shop history?
The software saves all transactions on your local computer hard drive. Back up this file to a separate location to prevent data loss.

Is there a limit to the number of items I can add?
You can add as many items as your hard drive can store. POS-fr handles large product catalogs without performance issues.

Can I track employee access?
The current version supports one administrator account. Future updates will include staff login features to track individual performance.
